О компании

Загорский трубный завод (ЗТЗ) — современное предприятие по производству труб большого диаметра для нефтегазовой отрасли. Завод оснащен передовым оборудованием и соответствует всем международным стандартам.

Продукция

Трубы для газопроводов

Диаметр: 530-1420 мм
Толщина стенки: 7-50 мм
Класс прочности: К52-К70
Стандарты: API 5L, ГОСТ 20295, ГОСТ 31447

Трубы для магистральных газопроводов производятся из высококачественной стали с учетом всех требований к прочности, износостойкости и безопасности. Способны выдерживать высокое давление и экстремальные условия эксплуатации.

Трубы для нефтепроводов

Диаметр: 530-1220 мм
Толщина стенки: 7-40 мм
Класс прочности: К52-К65
Стандарты: API 5L, ГОСТ 31443

Специализированные трубы для транспортировки нефти и нефтепродуктов. Обладают повышенной коррозионной стойкостью, имеют внутреннее антикоррозийное покрытие и высокую степень надежности.

Трубы для водопроводов

Диаметр: 530-1620 мм
Толщина стенки: 7-30 мм
Класс прочности: К48-К56
Стандарты: ГОСТ 10704, ГОСТ 30563

Трубы для водопроводов обеспечивают надежную транспортировку воды на дальние расстояния. Имеют специальное покрытие для защиты от внешних воздействий и увеличения срока службы.
